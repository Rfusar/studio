Crea il tuo primo server con Flask

Flask è una framework python per creare un server in modo semplice.

Ottimo per progetti di piccole e medie dimensioni. Puoi creare API anche se FastAPI sarebbe una scelta migliore, però anche Flask è un ottima scelta, siti statici, un po come questo :). Basta inizializzare flask (guarda il codice che seguo), aggiungere una secret-key per farsi che il server utilizzi le sessioni, cosi puoi personalizzare il sito in base all'utente loggato. Di default la cartella per l'HTML è templates, e per i fogli CSS e javascript è static.

          esempio di albero file:
          
          |__server.py
          |
          |__static/
             |
             |__css/
                |
                |__style.css
             |
             |__js/
                |
                |__main.js
          |
          |__templates/
             |
             |__index.html
